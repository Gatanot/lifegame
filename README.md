# Conway's Game of Life — ESP32-S3-RLCD-4.2

基于地形的自适应生命游戏，支持液压侵蚀、生物反馈和灾难事件，运行在 4.2 寸反射式LCD屏上。

## 硬件要求

| 组件 | 说明 |
|------|------|
| 开发板 | ESP32-S3-RLCD-4.2 |
| 屏幕 | 4.2" 反射式 LCD，400×300，SPI（MOSI=12, SCK=11, DC=5, CS=40, RST=41） |
| 按键 | BOOT（GPIO0）+ GP18（GPIO18），低电平有效，内部上拉 |
| PSRAM | **必须开启**（地形缓冲、LUT、网格等约 835KB） |

## 功能

- **六种地形** — Perlin 噪声 fBm 生成高程图（深海 → 高山），每种地形有独立的存活/出生规则，替代标准 B3/S23
- **液压侵蚀** — 水流累积、坡度阈值侵蚀、有机沉积反馈
- **相干抬升** — 双倍频 Perlin 抬升形成山脉
- **生物反馈** — 活细胞保护地形免于侵蚀；荒芜细胞加速侵蚀
- **灾难事件** — 火山喷发、塌陷、洪水入侵，约每 28 秒发生一次
- **GP18 按键** — 切换运行/暂停
- **BOOT 按键** — 重新播种地形噪声并生成新世界（每次按下地形不同）
- **Web 仿真器** — `web_simulator/index.html` 逻辑与 C++ 代码完全一致（浏览器直接打开）

## 编译烧录

1. 用 Arduino IDE 打开本文件夹
2. 选择开发板：**ESP32-S3 Dev Module**
3. 工具菜单开启 PSRAM（Octal SPI, 8MB）
4. 编译上传

## 按键操作

| 按键 | 功能 |
|------|------|
| GP18 单击 | 切换 运行/暂停 |
| BOOT 单击 | 重新生成地形 + 随机网格（保持暂停） |

## 文件结构

| 路径 | 说明 |
|------|------|
| `lifegame.ino` | 主程序 — 生命游戏逻辑、地形演化、主循环 |
| `display_bsp.cpp/.h` | RLCD 屏幕驱动 |
| `button_bsp.c/.h` | 按键 GPIO + IRQ（C → C++ 桥接） |
| `multi_button.c/.h` | 按键状态机（5ms 消抖，基于 `esp_timer`） |
| `perlin_noise.cpp/.h` | Perlin 噪声生成器（支持运行时重新播种） |
| `web_simulator/` | 浏览器端生命游戏仿真器（打开 `index.html`） |
